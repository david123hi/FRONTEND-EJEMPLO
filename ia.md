1. Generación de Estructura Base
Se solicitó a la IA la creación de un sistema de variables CSS (:root) basado en una paleta de colores específica (Morado y Dorado), estableciendo una base visual coherente desde el inicio.

2. Refactorización y "Corte" de Código
Para fines de documentación y aprendizaje, la IA ayudó a:

Simplificar el CSS: Eliminando estilos decorativos secundarios para dejar solo la estructura funcional.

Modularización: Separar la lógica del layout (flexbox) de los estilos de componentes (tarjetas e inputs).

3. Resolución de Dudas Técnicas (Git)
La IA actuó como consultor para interpretar el historial de Commits en VS Code, explicando conceptos como:

Outgoing Changes: Diferencia entre cambios locales y remotos.

Estado del Repositorio: Por qué desaparecen los archivos de la lista de cambios tras un commit exitoso.

4. Optimización de la Experiencia de Usuario (UX)
Se integraron consejos de la IA para mejorar la interactividad, sugiriendo el uso de:

transition para suavizar cambios de color.

outline: none con box-shadow personalizado para mejorar la accesibilidad en los inputs.
